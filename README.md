Objective : Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data.

Description :In this project, I built a Decision Tree Classification model using the Bank Marketing dataset. I started by loading the dataset and examining its structure,
checking for missing values, and analyzing the target variable.Since the dataset contained categorical variables, I used Label Encoding to convert them into numerical
values suitable for machine learning. The dataset was then divided into training and testing sets.I trained a Decision Tree Classifier using the entropy criterion and 
evaluated its performance using various metrics. Finally, I visualized the decision tree and identified the most important features influencing the model's predictions.

The project includes:
Loading and exploring the Bank Marketing dataset.
Checking dataset information and missing values.
Encoding categorical variables using Label Encoding.
Splitting the dataset into training and testing sets.
Training a Decision Tree Classifier.
Evaluating the model using Accuracy Score, Confusion Matrix, and Classification Report.
Visualizing the Decision Tree.
Identifying the top important features used by the model.

Technologies Used : 
Python
Pandas
NumPy
Matplotlib
Scikit-learn

Outcome : This project demonstrates how a Decision Tree Classifier can be used to solve a classification problem. It also highlights the importance of data preprocessing,
model evaluation, and feature importance analysis in building an effective machine learning model.
